## Image Files Needed

Add the following animated GIF files to this directory:

### Background
- **bg-starfield.gif** - Tiled starfield or geometric pattern background

### Decorative Elements
- **construction.gif** - Classic "Under Construction" animated GIF (for login page)
- **flame-divider.gif** - Animated flame divider bar (between sections)
- **sparkle.gif** - Sparkle/star animation (for revealed challenges)
- **spinning-star.gif** - Rotating star decoration

### Trophy/Awards
- **trophy.gif** - Animated trophy for leaderboard top ranks
- **medal-gold.gif** - Gold medal animation
- **medal-silver.gif** - Silver medal animation
- **medal-bronze.gif** - Bronze medal animation

**Note:** Sound effects are now stored in the `/audio` directory. See `/audio/README.md` for details.

## Where to Find GeoCities GIFs

- https://gifcities.org/ - Internet Archive's GeoCities GIF Search
- https://web.archive.org/web/20091027053343/http://geocities.yahoo.com/
- Search for "retro web gifs" or "90s web graphics"

## Usage in HTML

Once you add the GIF files, they're already referenced in the HTML:
- Login page: construction.gif
- Dashboard: flame-divider.gif, sparkle.gif
- Leaderboard: trophy.gif for top 3 ranks
